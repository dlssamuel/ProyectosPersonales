# 📂 Proyectos en Python – Portafolio

Este repositorio contiene diferentes proyectos desarrollados en **Python** como parte de mi aprendizaje y práctica en análisis de datos, programación orientada a objetos y machine learning.  
El objetivo es mostrar ejemplos claros, bien estructurados y comentados, listos para revisión.

---

## 📌 Proyectos incluidos

### 1. ✅ Gestor de Tareas
- Proyecto básico para gestionar tareas (añadir, marcar como completadas, listar pendientes).  
- Implementado con **clases en Python** para practicar programación orientada a objetos.  

---

### 2. 📊 Análisis de Datos con Regresión Logística
- Proyecto inspirado en un caso real de accidentalidad vial.  
- Incluye:
  - Preprocesamiento y limpieza de datos.
  - Separación en **train/test**.
  - Implementación de una clase `AlcoholLogisticModel` con métodos de:
    - Entrenamiento.
    - Predicción.
    - Evaluación (accuracy, precision, recall, F1-score).
    - Guardado y carga del modelo.

---

### 3. 📈 Regresión Lineal Básica
- Proyecto sencillo para entender los fundamentos de la **regresión lineal**.  
- Incluye generación de datos ficticios, ajuste de modelo y visualización de resultados.  

---

## 🛠️ Tecnologías y librerías utilizadas
- Python 3
- Pandas, NumPy
- scikit-learn
- Matplotlib, Seaborn
- Jupyter Notebook
